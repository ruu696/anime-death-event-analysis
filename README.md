# キャラクター死亡イベント分析

## 概要

本分析では、アニメ・漫画作品におけるキャラクターの死亡イベントが、
pixivの二次創作投稿数に与える影響を検証した。

## 分析の背景

私はアニメや漫画、小説などの創作文化が好きであり、
「キャラクターの死亡がファンの創作活動にどのような影響を与えるのか」
という疑問から本分析を行った。
